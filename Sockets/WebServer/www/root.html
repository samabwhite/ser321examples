You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /trig?len1=5&len2=3 -- returns information about the right triangle with inputted side lengths -- example link: http://18.118.7.128:9000/trig?len1=5&len2=3
  • /randomNumber?min=0&max=10 -- returns random integer value within the inputted range -- example link: http://18.118.7.128:9000/randomNumber?min=0&max=10
